# Teste de Git e GitHub

Este repositório contém códigos similares aos de um repositório anterior (Numero-Secreto-Avancado).

## Sobre o Projeto

Este repositório foi criado com o objetivo de testar funcionalidades do **Git** e analisar a interação entre o **Git** e o **GitHub**.

A proposta é utilizar este ambiente como laboratório para praticar comandos, versionamento, commits e integração com repositórios remotos.

---

## Objetivos

- Testar comandos básicos e avançados do Git
- Trabalhar com commits e histórico de versões
- Testar integração com o GitHub (push, pull, pull requests)
- Explorar boas práticas de versionamento

---

## Ferramentas Utilizadas

- Git
- GitHub
- Terminal (PowerShell / Git Bash)

---

## Comandos Testados

Alguns exemplos de comandos utilizados neste repositório:

- git init
- git clone
- git add .
- git commit -m "mensagem"
- git branch
- git pull
- git push
- git revert
- git reset
